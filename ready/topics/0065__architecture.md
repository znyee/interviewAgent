## API жҺҘеҸЈ

### дё»жҺҘеҸЈ
- `POST /api/chat` - жҷәиғҪеҜ№иҜқ (ж”ҜжҢҒжөҒејҸ)
- `POST /api/chat/stream` - жөҒејҸеҜ№иҜқ

### е·Ҙе…·жҺҘеҸЈ
- `POST /api/sql/query` - Text2SQLжҹҘиҜў
- `POST /api/code/execute` - д»Јз Ғжү§иЎҢ
- `POST /api/search` - зҪ‘з»ңжҗңзҙў
- `POST /api/rag/search` - RAGжЈҖзҙў

### ж•°жҚ®жҺҘеҸЈ
- `GET /api/stocks` - иӮЎзҘЁеҲ—иЎЁ
- `GET /api/stocks/{code}` - иӮЎзҘЁиҜҰжғ…
- `GET /api/financials/{code}` - иҙўеҠЎж•°жҚ®
- `GET /api/reports` - з ”жҠҘеҲ—иЎЁ
- `GET /api/schema` - ж•°жҚ®еә“з»“жһ„

## жҠҖжңҜж Ҳ

- **жЎҶжһ¶**: FastAPI + LangGraph
- **LLM**: qwen-max (йҳҝйҮҢдә‘)
- **Embedding**: text-embedding-v4
- **ж•°жҚ®еә“**: SQLite
- **PDFи§Јжһҗ**: PyPDF2 / DocMind
- **зҪ‘з»ңжҗңзҙў**: еҚҡжҹҘAPI
